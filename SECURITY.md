# Security Policy

Если вы нашли проблему безопасности, пожалуйста, не публикуйте её в открытом Issue.

Отправьте описание на:

info@amadeussinhro.com

Не отправляйте:

- пароли;
- ключи доступа;
- резервные копии базы данных;
- персональные данные пользователей;
- документы организаций.

Опишите:

- версию KOVCHEG;
- окружение сервера;
- шаги воспроизведения;
- ожидаемое и фактическое поведение.
