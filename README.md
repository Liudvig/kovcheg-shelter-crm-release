# 🐾 KOVCHEG Shelter CRM

Бесплатная CRM-система для приютов, волонтёрских организаций и проектов помощи животным.

- 🐕 учёт животных;
- 🏥 лечение и ветеринарные процедуры;

- ✅ задачи и дежурства;
- 👥 команда и роли;

- 🐾 ОСВВ;
- 🐕 кинология.

## Установка

Инструкция находится в разделе `docs`.

Перед установкой обязательно:

1. сделать резервную копию базы данных;
2. сделать копию файлов сайта;
3. проверить требования сервера.

## Тестирование

Если нашли ошибку — создайте Issue.

Укажите:

- версию KOVCHEG;
- PHP и сервер;
- браузер и устройство;
- описание проблемы;
- шаги воспроизведения;
- ожидаемый и фактический результат;
- скриншот или безопасный лог.

Не публикуйте:

- пароли;
- ключи доступа;
- персональные данные;
- дампы базы;
- конфигурацию сервера.
