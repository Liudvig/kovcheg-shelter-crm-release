# KOVCHEG Shelter CRM

Бесплатная CRM-система для приютов и организаций помощи животным.

## Статус проекта

Текущая версия: **Release Candidate**.

Проект открыт для тестирования сообществом. Мы собираем отзывы, исправляем ошибки и постепенно готовим стабильный релиз 1.0.

## Возможности

- учёт животных;
- лечение и ветеринарные процедуры;
- кормление и места содержания;
- склад и материальные потребности;
- задачи и дежурства;
- команда и роли;
- пристройство животных;
- доноры и финансы;
- документы;
- проекты и гранты;
- ОСВВ;
- кинология.

## Установка

Смотрите документацию в разделе `docs`.

Перед установкой обязательно сделайте резервную копию сервера и базы данных.

## Сообщить об ошибке

Используйте раздел Issues этого репозитория.

Пожалуйста, указывайте:

- версию KOVCHEG;
- PHP и версию сервера;
- браузер;
- описание проблемы;
- шаги воспроизведения;
- скриншоты и безопасные логи.

Не публикуйте пароли, ключи доступа, персональные данные и дампы базы данных.
