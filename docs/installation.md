# Установка KOVCHEG Shelter CRM

## Требования

- PHP 8.1 или новее;
- MySQL 8 или совместимая база данных;
- HTTPS рекомендуется.

## Перед установкой

1. Создайте резервную копию сервера.
2. Создайте новую базу данных.
3. Распакуйте релизный архив.
4. Запустите мастер установки.

## После установки

- создайте владельца системы;
- настройте роли;
- заполните данные организации;
- проверьте права доступа.

Для ошибок используйте Issues этого репозитория.
